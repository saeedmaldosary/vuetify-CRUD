# Vuetify CRUD

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Install JSON Server
```
npm install json-server -D
```

### Run JSON Server
```
json-server db.json
```
